## 🔴 В чем проблема?
После отсоединения CS:GO от Counter-Strike 2 была потеряна интеграция со Steam Workshop. Теперь при попытке зайти на сервер с кастомной картой, вы получаете ошибку.

## ✅ Наше решение
**Connectify** — это легкий лаунчер, который делает всю грязную работу за вас:
1.  **Считывает список серверов** прямо из Steam,
2.  **Проверяет карту:** если у вас её нет, программа качает её напрямую из Workshop,
3.  **Распаковывает:** автоматически отправляет файлы в папку игры,
4.  **Подключает:** запускает CS:GO и мгновенно соединяет вас с сервером.

<hr style="margin: 40px 0; border: 0; border-top: 1px solid #30363d;">

### 🛠 Системные требования
*   **ОС:** Windows 11 (64-bit),
*   **Игра:** установленная Counter-Strike: Global Offensive,
*   **И не более!**

### 🛡 Это безопасно?
Да. Программа работает как внешний помощник. Она **не инжектится** в процесс игры и не изменяет `.dll` файлы. Вы не получите VAC-бан за использование Connectify.
