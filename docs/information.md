### Роли и функционал:

#### Гость:
- Просмотр общего списка серверов с применением фильтров (по названию, наличию игроков, отсутствию пароля, статусу античита);
- Обновление списка серверов для получения актуальных данных о картах;
- Просмотр каталога игровых карт и поиск по их названию;
- Загрузка файлов карт (автоматически или в выбранную вручную директорию);
- Инициация подключения к серверу (при наличии карты — запуск игры, при отсутствии — предложение скачать карту).
#### Авторизованный пользователь:
- Весь описанный функционал гостя;
- Добавление серверов и карт в список «Избранное»;
- Управление личным кабинетом (изменение параметров запуска игры, обновление контактных данных);
- Просмотр истории последних подключений (до 20 серверов).
#### Администратор системы:
- Ведение каталога игровых режимов (категорий);
- Редактирование и актуализация реестра игровых карт (добавление, изменение описаний, удаление записей);
- Модерация списка серверов (выделение рекомендуемых/проверенных серверов, меток для карт);
- Просмотр статистики активности (количество скачиваний карт, популярность серверов);
- Управление учетными записями пользователей.

---

### Ограничения на ввод информации:

- **Авторизация:** пользователь не может добавить сервер в «Избранное» без авторизации;
- **Уникальность:** сервер не может быть добавлен в «Избранное» повторно;
- **Безопасность данных:** пароли пользователей хранятся только в зашифрованном виде (хеш-сумма);
- **Загрузка контента:** автоматическая загрузка блокируется до момента подтверждения пользователем согласия с объемом данных.;
- **Целостность данных:** система обязана проверять корректность пути к директории игры;
- **История:** история подключений ограничена 20 последними серверами для каждого пользователя;
- **Валидация:** игровой режим, добавляемый администратором, должен иметь отношение к реально существующим режимам.

### Определение сущностей и атрибутов:

**Сервер**:
- Уникальный IP-адрес и порт;
- Название сервера;
- Кодовое наименование текущей игровой карты;
- Текущее количество игроков;
- Максимальная вместимость;
- Сетевой отклик;
- Тип игрового режима;
- Флаги безопасности (пароль и статус античита);
- Дополнительные флаги (проверенный, рекомендованный).

**Игровая карта**:
- Уникальный идентификатор (Workshop ID);
- Полное наименование карты;
- Кодовое наименование;
- Дата создания/загрузки;
- Размер карты (МБ);
- Статус наличия (локальный файл).

**Пользователь**:
- Уникальное имя пользователя (логин);
- Адрес электронной почты;
- Хеш-сумма пароля;
- Дата регистрации;
- Путь к локальной директории игры;
- Персональные параметры запуска игры.

**Игровой режим (категория)**:
- Наименование режима (например: Surf, Deathmatch, Competitive).
